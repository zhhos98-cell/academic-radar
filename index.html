Academic Radar

Web Builder

Ready

Build a private radar profile in your browser.

Fill in your fields, feeds, and Bluesky sources. This page generates three plain-text files you can download and use with the local academic-radar command-line tool.

Privacy posture: this builder is static and client-side only. It has no analytics, cookies, accounts, uploads, backend API calls, or server-side storage.

Workflow

  1. 01

    Fill the form

    Add field terms, negative filters, RSS feeds, Bluesky queries, and watched accounts.

  2. 02

    Preview files

    Switch between JSON, Watchlist, and OPML to inspect exactly what will be downloaded.

  3. 03

    Download and run

    Save the three files, then run --summary, --doctor, and a dry preview locally.

Responsible use

Check sources and obligations before automation.

This builder only prepares local configuration files. Before running collection, review source terms, platform rules, institutional policies, and any data-protection duties that apply to your use case.

  • Keep private watchlists, feed exports, state files, and delivery settings out of public repositories.
  • Use dry runs and diagnostics before enabling email delivery or persistent state.
  • Do not use this project to collect or republish information you are not permitted to process.
